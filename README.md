# jeffrayshootsyou.com

The website for Jeff Ray, director / editor / cinematographer. Austin, TX.

## Stack
Static HTML + CSS + JS, served via `serve`. Deployed on Railway.

## Local dev
```
npm install
npm start
# open http://localhost:3000
```

## Pages
- `index.html` — homepage
- `portfolio.html` — full work catalog with click-to-play modal
- `services.html` — full pricing menu
